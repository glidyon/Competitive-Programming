# Text Elements
I like treating this problem as balancing the "array of colours".

In this problem, we basically need to check previous cell. If our current cell is
white, then we have to check if the previous cell is also white. If it is, it means, that
the current cell is not balanced at all. So we have to balance it by changing the current
cell to white colour. The same happens if our current cell is black,
and the previous one is black too. That means that we have to change our current cell to
white colour.
We have to write the change since the solution checks previous cells.

The time complexity of this problem is O(n), where n is the size of our string.
